### **문제 설명**

두 전봇대 A와 B 사이에 하나 둘씩 전깃줄을 추가하다 보니 전깃줄이 서로 교차하는 경우가 발생하였다. 합선의 위험이 있어 이들 중 몇 개의 전깃줄을 없애 전깃줄이 교차하지 않도록 만들려고 한다.

예를 들어, < 그림 1 >과 같이 전깃줄이 연결되어 있는 경우 A의 1번 위치와 B의 8번 위치를 잇는 전깃줄, A의 3번 위치와 B의 9번 위치를 잇는 전깃줄, A의 4번 위치와 B의 1번 위치를 잇는 전깃줄을 없애면 남아있는 모든 전깃줄이 서로 교차하지 않게 된다.

[](https://upload.acmicpc.net/d90221dd-eb80-419f-bdfb-5dd4ebac23af/-/preview/)

< 그림 1 >

전깃줄이 전봇대에 연결되는 위치는 전봇대 위에서부터 차례대로 번호가 매겨진다. 전깃줄의 개수와 전깃줄들이 두 전봇대에 연결되는 위치의 번호가 주어질 때, 남아있는 모든 전깃줄이 서로 교차하지 않게 하기 위해 없애야 하는 전깃줄의 최소 개수를 구하는 프로그램을 작성하시오.

### **입력**

첫째 줄에는 두 전봇대 사이의 전깃줄의 개수가 주어진다. 전깃줄의 개수는 100 이하의 자연수이다. 둘째 줄부터 한 줄에 하나씩 전깃줄이 A전봇대와 연결되는 위치의 번호와 B전봇대와 연결되는 위치의 번호가 차례로 주어진다. 위치의 번호는 500 이하의 자연수이고, 같은 위치에 두 개 이상의 전깃줄이 연결될 수 없다.

### **출력**

첫째 줄에 남아있는 모든 전깃줄이 서로 교차하지 않게 하기 위해 없애야 하는 전깃줄의 최소 개수를 출력한다.

# 분석

## 떠올리긴 했으나

- 가장 긴 수열을 찾으려곤 했음
  - 큐를 하나씩 만들어 붙이면 최장거리가 나오고, 그것 외에 다 제외하면 답이 나옴
  - 이까지는 인지했으나

### 중복되는 계산이 있음을 인지하기

- 결국 여러 번의 중복 계산이 포함됨.
  - **내가 필요한건 최댓값이지, 그 디테일한 요소가 아니다**
  - 이럴 때 쓰는게 DP이자 LIS 최장거리수열

## LIS

- 최장거리 수열의 **길이를 알고 싶을 때**
  - 어떤 구성인지는 알 필요 없음
- **마지막 인덱스를 기준으로 최장으로 연결되는 거리를 갱신해감**
  ```tsx
  for (let i = 0; i < n; i++) {
    for (let j = 0; j < i; j++) {
      if (lines[i][1] > lines[j][1]) {
        **dp[i] = Math.max(dp[i], dp[j] + 1);**
      }
    }
    lis = Math.max(lis, dp[i]);
  }
  ```
- i에서 끝나는 LIS를 구하려면? → 두 가지 중 하나
  1. **혼자만 있는 수열** → 길이 = 1
  2. **i보다 앞에 있는 어떤 j와 연결되는 수열**
     - 즉 arr[j] < arr[i]
     - **j에서 끝나는 LIS에 i를 붙이면 됨** → 길이 = dp[j] + 1

### 예시

arr = `[10, 20, 10, 30, 20, 50]`

초기 dp = `[1,1,1,1,1,1]` (자기 자신)

i=1 (arr[1]=20)

- j=0: arr[0]=10 < 20
  ⇒ dp[1] = max(1, dp[0]+1) = 2
  dp = `[1,2,1,1,1,1]`

---

i=2 (arr[2]=10)

- j=0: arr[0]=10 < 10 ❌
- j=1: 20 < 10 ❌
  dp = `[1,2,1,1,1,1]` (그냥 자기 자신)

---

i=3 (30)

- 10 < 30 → dp[3] = 2
- 20 < 30 → dp[3] = 3
- 10 < 30 → dp[3] = 3
  dp = `[1,2,1,3,1,1]`

---

i=4 (20)

- 10 < 20 → dp[4]=2
- 20 < 20 ❌
- 10 < 20 → dp[4]=2
  dp = `[1,2,1,3,2,1]`

---

i=5 (50)

- j=0: 10 < 50 → candidate = 2
- j=1: 20 < 50 → candidate = 3
- j=2: 10 < 50 → candidate = 2
- j=3: 30 < 50 → candidate = 4
- j=4: 20 < 50 → candidate = 3
  dp = `[1,2,1,3,2,4]`

---

최대값 = 4.
